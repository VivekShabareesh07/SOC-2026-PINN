# Fourier Neural Operator for 1D Burgers' Equation

This project implements a **Fourier Neural Operator (FNO)** to learn the solution operator of the 1D Burgers' equation. Instead of learning the solution for a single PDE instance like a PINN, the FNO learns the mapping from an initial condition \(u_0(x)\) to the corresponding solution \(u(x,T)\). The training data was generated by sampling smooth initial conditions using **Gaussian Random Fields (GRFs)** and numerically solving the Burgers' equation.

The model was implemented using **PyTorch** and the **neuraloperator** library. It achieved a **Relative L² Error of approximately 2.8%** on unseen test data. In addition to training and evaluation, the project includes experiments on parameter generalization and data efficiency, highlighting the strengths and limitations of neural operators for scientific machine learning.
